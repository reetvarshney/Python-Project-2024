Problem Definition 
In this project we are going to code a program for booking movie tickets, snacks, beverages and cancel tickets.
Overall, this code provides a comprehensive system for booking movie tickets and additional items, handling payments, and generating receipts for the user's transactions.


Datasets
In this project, datasets in tables are defined as dictionaries. There are in total 5 datasets in this project.
1.	Table for displaying menu of Beverages.
2.	Table for displaying menu of Snacks
3.	Table for displaying schedule for Movie1- Dil Chahta Hai
4.	Table for displaying schedule for Movie2- Jab we Met
5.	Table for displaying schedule for Movie3- Drishyam
